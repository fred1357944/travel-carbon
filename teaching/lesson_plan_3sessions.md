# 三堂導入教案 v0.1（永續素養取向）

**對象**：大學部（可含統計／跨系）  
**共授**：主任／統計系老師 + 工具導引  
**時間**：每堂 50–100 分鐘（可壓成工作坊）  
**資料**：**僅合成** `examples/sample_travel_records.csv`  

---

## 學習目標（對 DT／碳盤查敘事）

上完三堂，學生能夠：

1. 用自己的話說 Scope 1/2/3 與業務差旅（Cat.6）大概是什麼  
2. 指出 free-text 目的地為何讓「距離法」變難  
3. 操作開源工具產出距離與碳**篩檢**結果，並說出至少兩個**不確定性**來源  
4. 從「若我是永續單位」角度，談工具採納要什麼條件  

---

## 第 1 堂｜文件與歧義（可不裝軟體）

**導入（10）**  
- 輔大 114 永續素養通識：為什麼學校要你修永續？  
- 碳盤查不是只有電費：差旅也在 Scope 3  

**活動（30–40）**  
- 發合成「報帳事由」小卡（或 CSV 列印）  
- 小組標：台/臺、行政區、離島、中國城市、空值  
- 討論：若用「花錢多少」估碳，會出什麼問題？  

**收束（10）**  
- 預告下堂：用電腦把文字變成公里與 kg  

**作業**  
- 自願：寫一趟「虛構學術旅行」目的地（勿寫真人真事）  

---

## 第 2 堂｜上機操作

**Demo（15）**  
- CLI：`python -m travel_carbon 金門` / `日本 --profile taiwan_cfp`  
- 或 GUI 載入合成 CSV  
- 強調：`unknown` / 解析失敗不是略過，是要報告  

**實作（40）**  
- 每人或兩人一組跑 3–5 個目的地  
- 記錄：kind、km、screening vs taiwan_cfp  
- 填小表：哪個結果你敢給校長看？為什麼不敢？  

**收束（10）**  
- 係數與模式是假設，不是神諭  

**作業**  
- 上傳：一張輸出截圖 + 三句不確定性聲明  

---

## 第 3 堂｜批判、採用、素養

**分享（15）**  
- 選 2–3 組作業討論「工具可能低估／高估」  

**角色扮演（25）**  
- 角色：永續長／出納／教授／學生  
- 題：學校要不要全面導入此開源工具？條件？  

**設計回饋（20）**  
- 每人寫 3 個 UI／流程改進（可真的變成 GitHub issue）  

**收束**  
- 永續素養＝會用數字＋會懷疑數字  
- 與證照課（碳規劃等）的關係：概念認證 vs 手作識讀  

---

## 評量（低壓、可研究）

| 項目 | 權重示意 | 研究用途 |
|------|----------|----------|
| 第 2 堂操作表 | 必交 | 行為資料 |
| 不確定性三句 | 必交 | 質性 |
| 永續長 memo 半頁 | 必交 | 採用想像 |
| 課前課後 5–8 題 | 自願或加分 | 前後測 |

課前課後題方向（v0.1，正式前再修）：

1. Scope 3 是什麼（選擇題）  
2. 差旅碳比較常被放在哪個 Scope  
3. distance-based 與 spend-based 差在哪  
4. 自信程度：我能向非專業朋友解釋大學差旅碳  
5. 開放：工具最可能騙你的一點
