# 三堂導入教案 v0.2（統計系課嵌入 × 對齊永續素養框架）

**對象**：樂觀——**統計系課程**（洽談中；與主任／統計系老師共授）  
**定位（論文用句）**：*embedded in a statistics course, aligned with FJU’s sustainability-literacy GE reform (DT), not necessarily a DT-labelled course*  
**時間**：每堂 50–100 分鐘  
**資料**：**僅合成** `teaching/data/teaching_ledger_40.csv`（含合成核銷金額）  
**定案**：`docs/RESEARCH_DECISIONS.md`  

---

## 學習目標

1. 把 Scope 3／差旅碳說成**可測量的問題**（含 missing、分類誤差）  
2. 指出 free-text 目的地如何製造 **messy data**  
3. 操作工具產出距離與碳**篩檢**結果，並寫出至少兩個**不確定性**來源  
4. 從行政／校務研究角度，談「數字能不能進報告／dashboard」  

---

## 第 1 堂｜髒資料與測量（統計 hook）

**導入（10）— 不要開場「為何修永續」**  
- Hook：這是一份**真實形態**的行政文字資料（合成）。  
- 對統計學生：unknown 率 ≈ missing；係數選擇 ≈ measurement model；運具啟發式 ≈ 分類誤差。  
- 一句制度：學校同時在推永續素養與碳盤查敘事——你們練的是**把髒資料變成可稽核指標**。  

**活動（30–40）**  
- 發 `teaching_ledger_40.csv` 列印／投影  
- 小組標：台/臺、行政區、離島、國際、空值、**同目的地不同金額**  
- 討論：若用「核銷金額」估碳 vs 用「公里×係數」，偏差從哪來？  

**收束（10）**  
- 預告：下堂用開源工具跑完一輪 pipeline  

**作業**  
- 自願：虛構一筆學術旅行目的地（勿寫真人真事）  

---

## 第 2 堂｜上機操作

**Demo（15）**  
- CLI：`python -m travel_carbon 金門` / `日本 --profile taiwan_cfp`  
- 或 GUI 載入合成 CSV  
- 強調：`unknown` / 解析失敗不是略過，是要報告  

**實作（40）**  
- 每人或兩人一組跑 3–5 個目的地  
- 記錄：kind、km、screening vs taiwan_cfp  
- 填小表：哪個結果你敢給校長看？為什麼不敢？  

**收束（10）**  
- 係數與模式是假設，不是神諭  

**作業**  
- 上傳：一張輸出截圖 + 三句不確定性聲明  

---

## 第 3 堂｜批判、採用、素養

**分享（15）**  
- 選 2–3 組作業討論「工具可能低估／高估」  

**角色扮演（25）**  
- 角色：永續長／出納／教授／學生／**校務研究分析師**（要不要把數字放進 dashboard、放哪一層）  
- 題：學校要不要試用此開源工具？條件？正式報告能不能用？  

**設計回饋（20）**  
- 每人寫 3 個 UI／流程改進（可真的變成 GitHub issue）  

**收束**  
- 永續素養＝會用數字＋會懷疑數字  
- 與證照課（碳規劃等）的關係：概念認證 vs 手作識讀

## 評量（低壓、可研究）

| 項目 | 權重示意 | 研究用途 |
|------|----------|----------|
| 第 2 堂操作表 | 必交 | 行為資料 |
| 不確定性三句 | 必交 | 質性 |
| 永續長 memo 半頁 | 必交 | 採用想像 |
| 課前課後 5–8 題 | 自願或加分 | 前後測 |

課前課後題方向（v0.1，正式前再修）：

1. Scope 3 是什麼（選擇題）  
2. 差旅碳比較常被放在哪個 Scope  
3. distance-based 與 spend-based 差在哪  
4. 自信程度：我能向非專業朋友解釋大學差旅碳  
5. 開放：工具最可能騙你的一點
